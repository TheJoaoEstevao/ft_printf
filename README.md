# ft_printf

[Project Status](https://img.shields.io/badge/Project%20Status-100-green) ![Bonus](https://img.shields.io/badge/Project%20Status-100-green)

A custom implementation of the widely-used C standard library function `printf`. This `ft_printf` function replicates most of the functionality of the original `printf`.

## Features

- **Variadic Inputs**: Can take a variable number of arguments.
- **Format Specifiers**: Supports `%c`, `%s`, `%p`, `%d`, `%i`, `%u`, `%x`, and `%X`.
- **Width and Precision**: Control over the displayed results.
- **Flags Handling**: Adjusts the output formatting based on flags.
